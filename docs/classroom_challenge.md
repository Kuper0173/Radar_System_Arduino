# Build the Radar: Classroom Challenge

This activity is designed for students who receive the components and wires, then build the radar circuit step by step.

## Goal

Students should be able to:

- identify each component
- connect the component to the Arduino
- explain what each component does
- connect the physical circuit to electronics theory

## Suggested flow

1. Open `resources/radar_connection_challenge.html` in a browser.
2. Ask students to scroll through the cards one by one.
3. For each card, students should:
   - identify the real component
   - open the `Connections` tab
   - wire that component
   - open `How it works` if they want more depth
   - check the `Tips` tab before continuing
4. At the end, students complete the final checklist.
5. Upload the Arduino code and test the circuit.

## Theory topics included

The resource includes a small Theory Lab about:

- control systems
- ultrasonic sensing
- embedded systems
- signals and communication
- human-machine interface

These sections are intentionally short and didactic. They can be expanded later depending on the depth required for the class.
