Build the Radar

A simple connection challenge for an Arduino radar system. Students receive the components and wires, then build the circuit step by step.

How to use this resource

Scroll down and work with one component at a time. Each card includes the connections, a short explanation of how the component works, and a small tip. Click the tabs inside each card to change the information.

Theory Lab: what did you just build?

This radar is not only a wiring exercise. It is a tiny mix of engineering fields working together. Open each window to discover the ideas hiding inside the project.

🎮 Control systems

The joystick becomes a command

When you move the joystick, you are giving the Arduino a desired action. The Arduino reads that input and turns it into servo movement.

  • Input: joystick position.
  • Controller: Arduino code.
  • Output: servo angle.

Arduino Uno

The brain of the radar system.

Step 1: Arduino

Start by locating the pins that will receive information and control the other parts.

Main pins used:
D2 → Joystick SW
D9 → Servo signal
D10 → Ultrasonic Trig
D11 → Ultrasonic Echo
A0/A1 → Joystick axes
A4/A5 → OLED I2C

Breadboard

Organizes power, ground and signal wires.

Step 2: Breadboard

Use the breadboard to keep the circuit ordered and easier to debug.

Recommended organization:
Arduino 5V → + rail
Arduino GND → - rail
Signal wires → Arduino pins
Modules → stable rows
All GND connections must share the same ground.

Joystick

Controls the radar angle.

Step 3: Joystick

The joystick lets the student move the servo without using the Serial Monitor.

Connect the joystick:
GND → GND
5V → 5V
URX → A0
URY → A1
SW → D2

Micro Servo

Rotates from 0° to 180°.

Step 4: Servo

The servo moves the ultrasonic sensor to scan different angles.

Connect the servo:
Signal → D9
Red wire → 5V
Brown/Black wire → GND
If the servo shakes, use an external 5V supply and connect grounds together.

Ultrasonic

Measures distance using sound.

Step 5: Ultrasonic Sensor

This sensor detects how far an object is from the radar.

Connect the ultrasonic sensor:
VCC → 5V
GND → GND
Trig → D10
Echo → D11

OLED Display

Shows distance, angle and alerts.

Step 6: OLED Display

The display uses I2C communication, so it only needs two data wires.

Connect the OLED:
GND → GND
VCC → 5V
SDA → A4
SCL → A5
